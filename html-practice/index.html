Lists

Kevin Munoz | February 1, 2022

In HTML, we have two types of list ordered lists (<ol></ol>) and unordered lists (<ul></ul>). Each list item is determined using the li tag

An ordered list is a series of items that use a number to seperate each item. For example, an ordered list can be used to outline steps of a recipe. See examples below.

To do list

  1. Client meeting
  2. Email marketing campaign
  3. Audits

Favorite Singers

  1. Beyonce
  2. Doja Cat
  3. Ariana Grande

An unordered list is a series of times that use a bullet point to seperate each items where order wouldn't matter. For example, a list of groceries. See examples below.

Shopping List

  • Chicken (vegan and real)
  • Brussel Sprouts
  • Sweet Potato

Places I'd Like to Visit

  • India
  • Vietnam
  • Italy

Links

To add a link to your website you will need to use the <a> tag along with the href attribute. To set the href attribute, use href="" and add the destination of your link in the quotation marks.
